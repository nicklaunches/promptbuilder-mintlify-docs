## General

**What is Prompt Builder?**
Prompt Builder helps you create, refine, and reuse prompts across Generator, Optimizer, Assistant, and Library.

**What models can I use in Assistant?**
Assistant uses a curated set of models in the app. Pick the model you want to run your prompt with.

**Can I reuse prompts across modules?**
Yes. Save prompts to My Prompts and reuse them in Generator, Optimizer, and Assistant.

## Usage

**How do I start a new Assistant chat with a prompt?**
Use Run from Optimizer or Library, or paste a prompt directly into Assistant.

**Where are my prompts stored?**
All saved prompts live in My Prompts inside Prompt Library.

## Troubleshooting

**Why am I seeing an error?**
Check your connection, refresh, and try again. If it persists, contact support with details.
