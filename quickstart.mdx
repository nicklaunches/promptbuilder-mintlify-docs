## Get started in 5 steps

Get value fast: generate a first prompt, refine it, then save it to your Library for reuse.

<Info>
  **Open the app**: https://app.promptbuilder.cc
</Info>

<Steps>
  <Step title="Generate your first prompt">
    1. Open **Prompt Generator** in the app.
    2. Describe what you want to accomplish.
    3. Choose the **target AI model** (the prompt is optimized for this model).
    4. Click **Generate**.

    <Tip>
      Target model and chat model are separate: target model = optimized prompt, chat model = who helps you iterate.
    </Tip>
  </Step>

  <Step title="Refine it with chat">
    After generating, refine the prompt in the chat workspace:
    - “Make it shorter and more direct”
    - “Add constraints and acceptance criteria”
    - “Return output as JSON”
    - “Give 3 variations”

    When you like a version, click **Save** to store it in your Library.
  </Step>

  <Step title="Optimize an existing prompt">
    1. Open **Prompt Optimizer**.
    2. Paste an existing prompt (or choose one from **My Prompts**).
    3. Pick an optimizer model and click **Optimize**.

    The Optimizer saves a history entry automatically so you can compare versions later.
  </Step>

  <Step title="Save + reuse from the Library">
    Open **Prompt Library** to:
    - Find saved prompts in **My Prompts**
    - Pin your favorites
    - Run a prompt to start a new Assistant chat
  </Step>

  <Step title="(Optional) Generate social posts with SMM Bot">
    If you’re creating social content:
    1. Open **SMM Bot**
    2. Enter topic, tone, and audience
    3. Choose a platform (X, LinkedIn, Instagram, TikTok, Reddit)
    4. Click **Generate**, then refine in chat
  </Step>
</Steps>
