Prompt Builder helps developers standardize prompts for coding, debugging, and documentation.

## Common goals

- Improve code review and refactor prompts
- Create reliable debugging and repro workflows
- Standardize technical writing outputs

## A simple workflow

<Steps>
  <Step title="Start from an existing prompt">
    Use **Prompt Optimizer** to improve clarity and structure.
  </Step>

  <Step title="Run and iterate">
    Use **Prompt Assistant** to test responses and refine constraints.
  </Step>

  <Step title="Save the final prompt">
    Store it in **My Prompts** and pin it for future use.
  </Step>
</Steps>

## Example development workflows

- Code review checklists
- Bug reproduction instructions
- API documentation drafts
- Test plan templates
- Architecture summaries
