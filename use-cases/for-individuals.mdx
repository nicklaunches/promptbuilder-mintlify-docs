Prompt Builder helps individuals create reliable prompts for everyday tasks without starting from scratch each time.

## Common goals

- Draft content faster (emails, outlines, summaries)
- Standardize how you ask for outputs (format, tone, constraints)
- Build a personal library of reusable prompts

## A simple workflow

<Steps>
  <Step title="Generate your first draft">
    Use **Prompt Generator** to turn an idea into a structured prompt.
  </Step>

  <Step title="Refine it in Assistant">
    Run the prompt in **Prompt Assistant** and iterate on tone, length, or format.
  </Step>

  <Step title="Save for reuse">
    Save the final version to **My Prompts**, then pin your favorites.
  </Step>
</Steps>

## Example use cases

- Weekly status updates
- Job application cover letters
- Social post templates
- Research summaries
- Product feedback analysis
